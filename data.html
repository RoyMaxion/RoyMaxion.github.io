Keystroke Dynamics

Forensics Benchmark Data Set

This is both a benchmark data set for keystroke dynamics, and a supplement to the paper, "Comparing Anomaly-Detection Algorithms for Keystroke Dynamics," by Kevin Killourhy and Roy Maxion, published in conference. The paper is cited in the page that described the data set; it also available in the "Recent Publications" section of this web page.
